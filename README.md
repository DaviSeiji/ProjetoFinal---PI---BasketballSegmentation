# Basketball Segmentation — Projeto Final

Este repositório contém o projeto final da disciplina de **Processamento de Imagens**, cujo objetivo foi desenvolver e comparar diferentes métodos de segmentação para detectar automaticamente uma bola de basquete em imagens reais.

O trabalho envolveu análise, testes e comparação de diversas combinações de pré-processamento, segmentação e pós-processamento, com avaliação final por meio do coeficiente de Dice. Todas as imagens utilizadas já possuem máscaras reais, permitindo validação quantitativa dos resultados.

O relatório escrito está como pdf na página principal, com o nome de Projeto_Final__Processamento_de_Imagens.

Para testar basta rodar o arquivo main.py, que ele executa os testes, salva as melhores imagens e retorna uma lista da média dos dices.
